# Security Policy

## Supported Versions

This project is a single-version agentskills.io-compatible skill. The latest version on the `main` branch is the only supported version.

| Version | Supported          |
| :------ | :----------------- |
| latest  | :white_check_mark: |
| other   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this skill's behavior, reference files, or distribution mechanism, please
report it privately. Do not open a public issue.

Contact: **toolsoftrade.web@gmail.com**

You will receive an acknowledgement within 48 hours and a detailed response within 5 business days.
